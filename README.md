## `./bin/run info`

Display dev environment version information

```
USAGE
  $ ./bin/run info

OPTIONS
  -j, --json  output raw json

EXAMPLE
  $ aio info
```

## `./bin/run report`

Report an issue with the CLI

```
USAGE
  $ ./bin/run report

OPTIONS
  -b, --bug      report an issue
  -f, --feature  request a feature

EXAMPLE
  $ aio report
```
